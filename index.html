Detalhe dos Cartões em SPRINT BACKLOG

US-1: Cadastro de Anfitrião/Hóspede
Prioritário Médio | Sprint 1

Descrição: Como Anfitrião ou Turista, eu quero me cadastrar na plataforma SunHost Floripa para criar meu perfil e começar a usar os serviços.

Atividades Estratificadas:
  • [ ] Design da tela de cadastro (Web & Mobile)
  • [ ] Desenvolvimento do formulário (Frontend)
  • [ ] Integração com banco de dados (Backend)
  • [ ] Testes unitários (QA)
US-2: Listagem de Imóveis
Prioritário Médio | Sprint 1

Descrição: Como Anfitrião, eu quero listar meus imóveis na plataforma SunHost Floripa para que turistas possam encontrá-los e reservá-los.

Atividades Estratificadas:
  • [ ] Design da seção "Meus Imóveis"
  • [ ] Desenvolvimento do formulário de cadastro de imóvel
  • [ ] Upload e armazenamento de fotos
  • [ ] Testes de integração (QA)
US-3: Sistema de Reserva
Prioritário Médio | Sprint 1

Descrição: Como Turista Consciente, eu quero fazer uma reserva de acomodação selecionando as datas desejadas para confirmar minha estadia.

Atividades Estratificadas:
  • [ ] Design do calendário de disponibilidade
  • [ ] Desenvolvimento da tela de revisão de reserva
  • [ ] Lógica de bloqueio de datas (Backend)
  • [ ] Testes de carga (QA)
US-5: Filtros de Busca
Prioritário Médio | Sprint 1

Descrição: Como Turista Consciente, eu quero filtrar acomodações por datas, preço, tipo de imóvel e comodidades para encontrar a opção ideal para minha estadia.

Atividades Estratificadas:
  • [ ] Design da interface de filtros
  • [ ] Implementação de busca em tempo real
  • [ ] Otimização de queries de banco de dados
  • [ ] Testes de performance (QA)
US-7: Notificações e Alertas
Próxima Versão Médio | Sprint 2

Descrição: Como Anfitrião ou Turista, eu quero receber notificações em tempo real sobre solicitações de reserva, confirmações, cancelamentos e mensagens para manter-me informado.

Atividades Estratificadas:
  • [ ] Design do sistema de notificações (Arquitetura)
  • [ ] Desenvolvimento de serviço de notificações (Backend)
  • [ ] Implementação de push notifications
  • [ ] Testes unitários (QA)

Este quadro simula o layout e o conteúdo de uma ferramenta Kanban (como Notion ou GitHub Projects), contendo todas as informações e priorizações solicitadas.
